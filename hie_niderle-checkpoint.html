Analýza stupně myelinizace u novorozenců pomocí T1w/T2w poměru¶

Tento notebook shrnuje postup zpracování a analýzy MRI dat. Cílem je objektivně změřit úroveň myelinizace v oblastech bazálních ganglií, thalamu a hipokampu. Postup se skládá ze 4 hlavních kroků:

  1. Vizuální kontrola vstupních dat (QC 1): Automatizované otevírání T1w a T2w snímků v programu ITK-SNAP pro zhodnocení kvality. Výstupem je textový seznam schválených pacientů pro další zpracování.
  2. Prostorová registrace a výpočet poměru: Zarovnání T2w snímků na anatomické T1w. Metodika se inspiruje prací Soun et al. (2017) pro hodnocení myelinizace. Na rozdíl od původní studie neprovádíme normalizaci na referenční tkáně (sval/oko), ale pracujeme s přímým voxel-wise poměrem T1w/T2w, ze kterého bereme statisticky robustní medián.
  3. Kontrola zarovnání masek (QC 2): Resampling FreeSurfer segmentací (z rozlišení 256x256x256) zpět do originální geometrie snímků pacienta. Následuje druhá vizuální kontrola přesnosti kontur v ITK-SNAP.
  4. Extrakce dat a vizualizace: Výpočet mediánu T1w/T2w pro zvolené oblasti zájmu (POI). Export datových matic a jejich vizualizace pomocí krabicových a interaktivních (Plotly) grafů pro snadnou identifikaci odlehlých hodnot (potenciální HIE patologie).

1. Kontrola kvality vstupních dat (QC 1)¶

Tento skript načítá surové T1w a T2w snímky a postupně je otevírá v programu ITK-SNAP. Cílem je manuálně zhodnotit, zda data neobsahují pohybové artefakty a jsou vhodná pro analýzu. Výstupem tohoto bloku je textový soubor se seznamem schválených pacientů, který slouží jako filtr pro další zpracování.

2. Předzpracování obrazu, prostorová registrace a výpočet T1w/T2w poměru¶

V tomto kroku probíhá klíčová příprava dat, která se metodicky opírá o postupy ze studií Soun et al. a Ganzetti et al. Proces je rozdělen do tří na sebe navazujících fází:

  • Korekce nehomogenity (Bias Field Correction): Magnetická rezonance nikdy nesnímá zcela rovnoměrně, což vytváří na snímcích pozvolné změny jasu ("stíny"). Při přímém dělení T1w a T2w signálu by tyto odchylky vytvořily na výsledné mapě masivní falešné artefakty. Proto jako první krok aplikujeme na oba snímky algoritmus N4BiasFieldCorrection, který tyto stíny matematicky odstraní.
  • Prostorová registrace (Lícování snímků): T2w snímek je následně zarovnán na anatomický T1w snímek pacienta. Jelikož lícujeme snímky téhož mozku, využíváme pouze lineární rigidní transformaci (Rigid-body) v knihovně SimpleITK. Snímek je pouze posouván a rotován, čímž se vyhneme nelineárním deformacím, které by mohly poškodit původní anatomická data.
  • Standardizace intenzity a výpočet poměru: Vzhledem k patologickým změnám u pacientů s HIE nelze pro kalibraci jasu použít samotnou mozkovou tkáň. Zmíněné studie tento problém řešily normalizací na externí tkáně (sklivec či žvýkací sval). Automatická segmentace těchto oblastí u novorozenců je však vysoce riziková a náchylná k chybám (např. kvůli oříznutí tváře na skenech). Namísto riskantní externí kalibrace jsem proto zvolil robustnější matematický přístup (např. vyrovnání histogramů / z-skóre normalizaci) na surových datech. Teprve z takto ošetřených dat je vypočítán voxel-wise poměr T1w/T2w, ze kterého pomocí FreeSurfer masek extrahujeme statisticky odolný medián.

import os
import SimpleITK as sitk

# 1. NASTAVENÍ CEST
seznam_souboru = "vybrane_snimky.txt"
vystupni_slozka_hlavni = r"D:\bakalarka\data_hie\derivatives\coregistered"

if not os.path.exists(vystupni_slozka_hlavni):
    os.makedirs(vystupni_slozka_hlavni)

# 2. POMOCNÉ FUNKCE (N4 a Registrace)
def odstran_stiny_n4(image):
    """Provede N4 Bias Field Correction pro vyhlazení jasu."""
    image_float = sitk.Cast(image, sitk.sitkFloat32)
    maska = sitk.OtsuThreshold(image_float, 0, 1, 200)
    corrector = sitk.N4BiasFieldCorrectionImageFilter()
    return corrector.Execute(image_float, maska)

def registruj_t2_na_t1(fixed_image, moving_image):
    """Provede rigidní registraci T2w (moving) na T1w (fixed)."""
    initial_transform = sitk.CenteredTransformInitializer(
        fixed_image, moving_image, 
        sitk.Euler3DTransform(), 
        sitk.CenteredTransformInitializerFilter.GEOMETRY
    )
    
    registration_method = sitk.ImageRegistrationMethod()
    registration_method.SetMetricAsMattesMutualInformation(numberOfHistogramBins=50)
    registration_method.SetMetricSamplingStrategy(registration_method.RANDOM)
    registration_method.SetMetricSamplingPercentage(0.01)
    registration_method.SetInterpolator(sitk.sitkLinear)
    
    registration_method.SetOptimizerAsGradientDescent(
        learningRate=1.0, numberOfIterations=100, 
        convergenceMinimumValue=1e-6, convergenceWindowSize=10
    )
    registration_method.SetOptimizerScalesFromPhysicalShift()
    registration_method.SetInitialTransform(initial_transform, inPlace=False)
    
    final_transform = registration_method.Execute(fixed_image, moving_image)
    
    resampler = sitk.ResampleImageFilter()
    resampler.SetReferenceImage(fixed_image)
    resampler.SetInterpolator(sitk.sitkBSpline) 
    resampler.SetDefaultPixelValue(0)
    resampler.SetTransform(final_transform)
    
    return resampler.Execute(moving_image)

# 3. HLAVNÍ CYKLUS VÝPOČTŮ
if not os.path.exists(seznam_souboru):
    print(f"CHYBA: Soubor {seznam_souboru} neexistuje. Spusť nejprve QC skript nahoře!")
else:
    with open(seznam_souboru, "r") as f:
        radky = f.readlines()
        
    print(f"Načteno {len(radky)} pacientů z textového souboru. Začínáme výpočty!\n")
    
    for radek in radky:
        radek = radek.strip()
        if not radek: continue
            
        pacient, t1_cesta, t2_cesta = radek.split(',')
        print(f"--- Zpracovávám pacienta: {pacient} ---")
        
        pacient_out_dir = os.path.join(vystupni_slozka_hlavni, pacient)
        if not os.path.exists(pacient_out_dir):
            os.makedirs(pacient_out_dir)
            
        print("  -> Načítám snímky...")
        t1_img = sitk.ReadImage(t1_cesta)
        t2_img = sitk.ReadImage(t2_cesta)
        
        print("  -> Odstraňuji stíny (N4 Bias Correction - chvíli to potrvá)...")
        t1_clean = odstran_stiny_n4(t1_img)
        t2_clean = odstran_stiny_n4(t2_img)
        
        print("  -> Zarovnávám T2w na T1w (Rigidní registrace)...")
        t2_registered = registruj_t2_na_t1(t1_clean, t2_clean)
        
        print("  -> Ukládám hotové snímky...")
        t1_clean = sitk.Cast(t1_clean, sitk.sitkInt16)
        t2_registered = sitk.Cast(t2_registered, sitk.sitkInt16)
        
        sitk.WriteImage(t1_clean, os.path.join(pacient_out_dir, f"{pacient}_T1w_N4.nii.gz"))
        sitk.WriteImage(t2_registered, os.path.join(pacient_out_dir, f"{pacient}_T2w_reg.nii.gz"))
        
        print(f"  [HOTOVO] Pacient {pacient} byl úspěšně zarovnán a uložen.\n")

    print(f"VŠECHNO HOTOVO (KROK 2)! Zkontroluj si nová data ve složce: {vystupni_slozka_hlavni}")

import os
import SimpleITK as sitk

coreg_dir = r"D:\bakalarka\data_hie\derivatives\coregistered"

# Najdeme všechny složky pacientů
pacienti = [d for d in os.listdir(coreg_dir) if d.startswith("sub-")]

for pacient in pacienti:
    print(f"Zpracovávám masku pro: {pacient}")
    
    # Tvé přesné f-string cesty
    cesta_t1 = fr"D:\bakalarka\data_hie\derivatives\coregistered\{pacient}\{pacient}_T1w_N4.nii.gz"
    cesta_maska = fr"D:\bakalarka\data_hie\derivatives\infantfs-8.1.0\{pacient}\mri\aseg.nii.gz"
    vystup_kontrola = fr"D:\bakalarka\data_hie\derivatives\coregistered\{pacient}\kontrola_masky.nii.gz"
    
    # Rychlá pojistka: zkontrolujeme, jestli zdrojové soubory existují
    if not os.path.exists(cesta_t1) or not os.path.exists(cesta_maska):

    # Načtení a přepočet

    sitk.WriteImage(maska_resampled, vystup_kontrola)
    print("  [+] Maska úspěšně uložena.")

print("\nHOTOVO! Všechny masky jsou připravené.")
